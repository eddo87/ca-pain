# World Map

[Wiki page](https://github.com/Reetus/ClassicAssist/wiki/World%20Map) · 3 commands

- **AddMapMarker(name, x, y, facet, [zoomlevel], [iconname])**
  Adds a marker on the world map
  `name`: String representing a name, ie "Snoopy". · `x`: X Coordinate. · `y`: Y Coordinate.
  `AddMapMarker("Treasure", 1000, 1000, 0)`

- **ClearMapMarkers()**
  Clears all map markers
  `ClearMapMarkers()`

- **RemoveMapMarker(name)**
  Removes all map markers with the specified name from the world map
  `RemoveMapMarker("Treasure")`
